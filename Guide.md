# LuminusCampus Overview

LuminusCampus is a web-based ERP system for managing various school/office modules. It handles things like student records, staff, fees, and more.

### Key Info
*   **Tech:** ASP.NET Core 8.0, Entity Framework Core (MySQL).
*   **Core Logic:** Role-based access (SuperAdmin, Admin, Staff).
*   **Database:** `Testing1` (MySQL).
*   **URL Setup:** Uses a simple obfuscation system to keep internal URLs private.
